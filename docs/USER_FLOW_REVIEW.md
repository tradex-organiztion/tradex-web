## 2. 누락된 플로우

### 2.1 Login & Signup

| 누락 항목 | 위치 |
|----------|------|
| 비밀번호 찾기 | 로그인에서 분기 |

```
로그인
├── 소셜 로그인 ───→ ...
├── 기본 로그인 ───→ ...
└── 비밀번호 찾기 ──→ 이메일 입력 → 재설정 메일 발송 → 비밀번호 변경    [누락]
```

---

### 2.2 수신함

| 누락 항목 | 위치 |
|----------|------|
| 알림 유형 분류 | 수신함 하위 |

현재 "차트 관련 알림"만 표시되어 있음. 아래 3가지 유형으로 분류 필요.

```
수신함
├── 트리거 알림 ────→ 차트 바로가기
├── 매매원칙 알림 ──→ 매매일지 바로가기    [누락]
└── 시스템 알림 ────→ 읽음 처리            [누락]
```

---

### 2.3 매매 등록

| 누락 항목 | 위치 |
|----------|------|
| 매매 원칙 삭제 | 매매 등록 하위 |

```
매매 등록
├── AI 매매 원칙 추천
├── 매매 원칙 추가
├── 매매 원칙 수정
└── 매매 원칙 삭제    [누락]
```

---

### 2.4 매매일지 관리

| 누락 항목 | 위치 |
|----------|------|
| 매매일지 수정 | 매매일지 관리 하위 |
| 매매일지 삭제 | 매매일지 관리 하위 |

```
매매일지 관리
├── 매매기록 추가하기
├── 매매일지 작성하기
├── 매매일지 수정하기    [누락]
└── 매매일지 삭제하기    [누락]
```

---

### 2.5 차트분석

| 누락 항목 | 위치 |
|----------|------|
| 트리거 삭제 | Trading System 하위 |

```
Trading System
├── 설정된 트리거 확인
├── 트리거 식별하기
└── 트리거 삭제하기    [누락]
```

---

### 2.6 설정

| 누락 항목 | 위치 |
|----------|------|
| 거래소 API 연동 해제 | 계정 설정 하위 |
| 회원탈퇴 | 계정 설정 하위 |

```
계정 설정
├── 프로필 사진
├── 닉네임
├── 비밀번호
├── 거래소 API
│   ├── API 연동
│   └── API 연동 해제    [누락]
├── 로그아웃
└── 회원탈퇴             [누락]
```

## 3. 누락된 분기

다이어그램에서 Yes/No 또는 성공/실패 분기가 필요한 항목.

| 위치 | 누락된 분기 |
|------|------------|
| 거래소 API 연동 | 연동 테스트 → 성공/실패 |
| 구독 설정 → 결제하기 | 결제 → 성공/실패 |
| 로그인 | 로그인 → 성공/실패 |
| 회원가입 | 가입 → 성공/실패 |

## 4. 하위 항목 추가 필요

현재 단일 박스로 표현되어 있으나, 하위 항목 상세화가 필요한 영역.

### 4.1 Login & Signup

| 대상 | 현재 | 추가 필요 항목 |
|------|------|---------------|
| 추가 정보 입력 | 단일 박스 | 닉네임 입력, 투자 경험 선택, 관심 자산 선택 |

```
추가 정보 입력
├── 닉네임 입력
├── 투자 경험 선택 (초보/중급/고급)
└── 관심 자산 선택 (주식/암호화폐/선물)
```

---

### 4.2 홈

| 대상 | 현재 | 추가 필요 항목 |
|------|------|---------------|
| 홈 위젯 | AI 분석, 수익, 리스크 매핑 | 최근 알림, 최근 매매일지 |

```
홈
├── AI 분석 보러가기
├── 수익
├── 리스크 매핑
├── 최근 알림 (3건)        [추가]
├── 최근 매매일지 (3건)    [추가]
└── Tradex AI(Side)
```

## 5. 권장 사항

### 페이지 경로 표기

각 화면 박스에 실제 URL 경로를 함께 표기하면 개발 시 참고하기 용이함.

**예시**
```
┌──────────────────────┐        ┌──────────────────────┐
│ 매매일지              │   →    │ 매매일지              │
│                      │        │ /trading/journal     │
└──────────────────────┘        └──────────────────────┘

┌──────────────────────┐        ┌──────────────────────┐
│ 전략 분석            │   →    │ 전략 분석             │
│                      │        │ /analysis/strategy   │
└──────────────────────┘        └──────────────────────┘
```
